## Principles

Create Your UI Component Library.<br/>
Stlato UI is a simple UI Component Library Kit.<br/>
Fork it, customize it, and create your own so easily!

## overviews

```jsx
import { Button } from "stlato-ui";

const YourApp = () => {
  return <Button>Hi There !</Button>;
};
```

## features

- Full Customize
- TypeScript First Class Support

And lot [more](/docs/getting-started).

## Getting Started

### Fork(recommended)

1. Fork repository stlato UI
2. Customize UI
3. Use it

package.json of your frontend application.

```bash
"dependencies": {
  "package-name": "git+https://<github_token>:x-oauth-basic@github.com/<user>/<your-foked-repo>.git"
}
```
